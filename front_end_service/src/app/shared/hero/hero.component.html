HippoNuage, c'est quoi?

HippoNuage, c'est une solution de cloud, pensée & développée par deux élèves étudiant en développement fullstack

Conçue en microservices, cette solution leur permet de développer de solides compétences en web.

Spring-Boot pour les trois API / Angular pour une SPA optimale & scalable, un système de stockage sur API tierce (S3) et une base de données PostGreSQL, le tout conteneurisé avec docker
